Get Started in Minutes

Download the latest release or build from source.

1

Download & Install

Grab the latest self-contained Windows x64 executable from GitHub Releases. No runtime installation required.

2

Register an App

Create an Azure AD app registration in your tenant with the required Microsoft Graph delegated permissions. Set the redirect URI to http://localhost:45132.

3

Add a Profile

Launch the app, enter your Tenant ID and Client ID, choose your cloud environment, and save. Sign in with your work account ‚Äî you're ready to go.
